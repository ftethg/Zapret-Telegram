## 3-Способ-проверки

```
Вы тут оказалить по несчастью либо у вас работает ютуб без впн (
Для начала надо в папку с программой зайти в папку lists 
Путь примерно такой zapret\lists
В данной папке нам нужен файл list-general.txt
Нажимаем по нему правой кнопкой мыши и открываем через блокнот
```
